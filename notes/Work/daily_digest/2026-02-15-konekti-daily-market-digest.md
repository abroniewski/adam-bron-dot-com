## 3) Learning Corner

### What is a control tower?
A control tower is a real-time operational layer that combines cross-system data, surfaces deviations, and supports intervention decisions. It should help teams act, not only observe.

### What is an accelerator?
An accelerator is a reusable package of assets (for example data model patterns, KPI logic, integration components, and playbooks) that shortens time-to-value and reduces implementation risk.

### What does "building a new process mining use case" actually involve?
At minimum:
1. Define the decision to improve and the accountable owner.
2. Define process/object scope.
3. Connect and validate data.
4. Set baseline KPIs.
5. Detect and prioritise deviations.
6. Tie insights to interventions and value tracking.

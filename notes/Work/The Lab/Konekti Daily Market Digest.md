# Konekti Daily Market Digest

## Objective
Produce a daily strategic digest for leadership and commercial action in enterprise AI, process mining, and adjacent markets.

## How it works
1. Run daily source scan.
2. Filter candidates for strategic relevance.
3. Synthesize into one coherent storyline.
4. Add stakeholder impact and pipeline actions.
5. Include competitor and category watch.
6. Publish in in-email friendly format.
